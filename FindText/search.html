Search fields

Combine the clues you already know.

Use one field for a broad search or combine fields to narrow a large folder quickly. Search up to three content keywords together, with different highlight colors helping each term stand out in results and previews.

Four fields

Each field narrows the result set in a different way.

Search fields work independently or together. Use the file name field when you remember part of the name, the extension field to filter by format, the content field to search inside documents, and the exclude field to remove unwanted matches.

File Name

Search for words you remember from the file name, such as a client, project, year, or document title.

Example: type contract 2026 when you remember both the document type and year.

Extension

Limit results to the formats you care about, such as PDFs, Word documents, spreadsheets, presentations, or text files.

Example: use pdf docx when you only want PDF and Word results.

Content

Search for words or phrases inside documents, even when the file name does not tell you enough.

Search up to three content keywords together. Each keyword can be highlighted in a different color, making it easier to scan the matching passage.

Exclude

Remove documents that contain specific words. Useful for filtering out drafts, templates, or archived copies.

Example: excluding draft removes any document whose content includes the word "draft".

Combined search

Use all fields together for precision.

When you know the file name, the format, and a phrase inside the document, combining fields gives you the fastest path to the right file.

Search style

Focused when you know the details, flexible when you do not.

Use several clues together when you know what you are looking for, or search more broadly when you only remember one part of the document.

Focused search

Combine file name, format, and content clues to narrow a large folder to the most relevant documents.

Broader discovery

Start with a single term or format when you are exploring an archive and want to see what is available.

Preview

See matching text before you open the file.

When you select a search result, the preview pane shows the matching passage with highlighted keywords. Up to three searched terms can be visually separated with different colors.

Keyword highlights

Up to three matching keywords can be highlighted with different colors, making it easy to scan the passage and verify that the document contains what you need.

Navigate matches

Use the match navigation controls to jump between highlighted keywords within the preview, and between matched pages across longer documents.
