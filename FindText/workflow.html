Workflow

Choose folders, build the index, search content.

Find Text turns selected Mac folders into a searchable workspace, so you can find the right document by name, type, or words inside the file.

Three steps

From folders to search results in minutes.

Each step is designed to be straightforward. Add the folders you want to search, let the indexer process your documents, then use the search fields to find what you need.

Add folders

Choose one or more Mac folders, including cloud-synced folders that are available locally through Finder. Each folder is authorized through macOS security permissions.

You control which locations are indexed. Add project folders, document archives, or any Finder-accessible directory.

Build or update the index

Let Find Text prepare your selected folders for fast local search. The app reads supported documents and keeps the searchable information on your Mac.

Refresh when files are added, removed, renamed, or changed. Automatic updates are available with Pro.

Search and preview

Search across file name, file type, content, and exclude fields. Results appear quickly, and you can preview up to three content keywords with different highlight colors.

Click any result to see the matching passage in the preview pane before opening the file.

App preview

See the workflow in action.

The interface shows your folder list on the left, search fields at the top, matching results in the center, and a document preview on the right.

Designed for real folders

Fast search without digging through nested folders.

Find Text is made for document-heavy folders where opening files one by one wastes time. Search first, preview the match, then open only what matters.

Good for large collections

Use Find Text with project archives, client folders, research notes, invoices, manuals, and other document libraries that grow over time.

Easy to keep current

When a folder changes, refresh the search data and continue working with an up-to-date view of your files.

Everyday actions

Move from search result to document quickly.

After you find a result, open it, reveal it in Finder, or copy its path for sharing and follow-up work.

Preview before opening

Select a result to inspect the matching passage first, so you can confirm it is the right file before opening the full document.

Open or reveal instantly

Use the context menu to open the file, choose another app, reveal the location in Finder, or copy the full file path.
