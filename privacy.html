Privacy Policy

Quick Search File is built around local search. The app indexes folders you choose so you can find files by filename, extension, folder path, and search terms.

Last updated: May 16, 2026

Overview

This Privacy Policy explains how Quick Search File handles information in the iOS app, macOS app, optional Command Line Interface (CLI), Google Drive read-only features, and this website.

Quick Search File is designed to keep search data local whenever possible. The app does not sell personal information, does not use your file data for advertising, and does not use your Google Drive data to train artificial intelligence or machine learning models.

Information the app processes

When you authorize folders or connect a supported storage provider, the app may process filenames, file extensions, folder paths, file identifiers, file sizes, modified dates, selected folder information, search terms, index update status, app settings, subscription status, and related metadata needed to display search results.

The app is focused on filename and metadata search. It does not upload your local search index to a Quick Search File server.

Google Drive read-only access

If you choose to connect Google Drive, Quick Search File uses Google OAuth and the https://www.googleapis.com/auth/drive.readonly permission scope. This scope allows the app to view Google Drive files and folders that your Google account can access.

Quick Search File uses Google Drive access to list and read file and folder metadata needed for search, indexing, result display, and user-directed file access. The app does not use Google Drive access to create, edit, move, delete, or share your Google Drive files.

You can revoke Google Drive access at any time from your Google Account security settings. You can also remove indexed data by deleting the app, clearing app data where available, or removing the connected account inside the app when that option is available.

Google API Limited Use disclosure

Quick Search File's use and transfer of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

Google user data is used only to provide or improve user-facing file search features. It is not sold, used for advertising, transferred for unrelated purposes, or used to train generalized AI or machine learning models.

Local indexes and storage

Search indexes are stored on your device. On iOS, folder access is controlled by the system Files picker and the permissions you grant. On macOS, folder access is controlled by macOS permissions, app settings, and any connected storage providers you choose to use.

If the macOS Command Line Interface (CLI) is enabled, it uses the same app-authorized local index. CLI output may include filenames, paths, extensions, file sizes, and other metadata you request in text, paths-only, or JSON form.

Subscriptions and App Store data

Purchases and subscriptions are handled by Apple through the App Store. Quick Search File may receive subscription status needed to unlock app features, but payment details are processed by Apple.

Website data

This website is a static product website hosted on GitHub Pages. Standard server logs or platform-level analytics may process basic request information such as IP address, browser, referring page, and request time for security, reliability, and hosting operations.

Data sharing

Quick Search File does not sell your personal information. Data may be shared only when needed to provide user-requested functionality, comply with law, protect security, process App Store subscriptions through Apple, operate hosting infrastructure, or when you explicitly export, copy, open, or share information yourself.

Data retention and deletion

Local index data remains on your device until you remove indexed folders, clear app data where available, delete the app, or use an in-app removal option. Google Drive access can be revoked through your Google Account permissions.

Children

Quick Search File is not directed to children under 13. If you believe a child has provided personal information, please contact support so it can be addressed.

Changes

This policy may be updated as features change, including changes to Google Drive support, storage providers, subscriptions, or platform requirements. The latest version will be posted on this page.

Contact

For support and privacy questions, use the Quick Search File support page at https://quicksearchfile.com/support.html.
